## Technology:
- Tool: Selenium
- IDE: IntelliJ IDEA Community Edition
- Build Tool: Gradle
- Language: Java
- Test Runner: TestNG

## Prerequisite:
1. JDK 1.8 must be installed.
2. Requires reliable internet access

## Run The Automation Script
```sh
Terminal: gradle clean test
```
